Basic setup and API usage documentation

Registration

Registration is a simple 2 step process.

  • First, register as a user on the Baresniff platform.
  • Second, register your app to be able to use the Baresniff APIs. Each client accesses the APIs via OAuth which provides a convenient and secure method of access.

Customize

During registration of the app/client you must customize the Baresniff APIs to suite your specific needs. There are a number of data fields to choose from - ShortTextField, LongTextField, DateTimeField, GeoLocationField, URLField, etc.
